Building Your First Watch

In this guide, let's use TickTock to create our first animated watch. We will be building the watch shown in the above graphic. It contains two dials, the second being set to a specific time zone and a power reserve indicator.

The SVG has been created already and the optimized code can be downloaded below. This is all we need to get started.

Download Optimized SVG Code

index.html

Let's take the downloaded SVG code and drop that into our index.html file. Once this is opened in our browser, we should see our watch in its static form. Feel free to wrap the watch in a container or apply sizing styles to the SVG to keep it from filling the entire screen.

Next, before digging into animation and TickTock settings, let's first install the library. Open a terminal and in your working directory install TickTock with whichever manager you prefer. 

yarn add ticktock-js --dev

With TickTock installed, let's create and dig into an app.js file.

app.js 

(function() {
  "use strict"

  const Watch = require('ticktock-js');
  const settings = { ... };

  let firstWatch = new Watch(settings);
})();

We first import TickTock into our file and assign it to our constant Watch variable. From there we can call new Watch() to instantiate a new watch with whatever settings we wish. Let's dig into our settings next.

We know we have two dials, a primary and secondary. So let's start there. 

const settings = {
  dials: [{
    name: 'primary',
    hands: {
      hour: 'primary-hour-hand',
      minute: 'primary-minute-hand',
      second: 'primary-second-hand'
    }
  },
  {
    name: 'secondary',
    hands: {
      hour: 'secondary-hour-hand',
      minute: 'secondary-minute-hand',
      second: 'secondary-second-hand'
    }
  }]
};

Note the importance of the IDs that are defined in the SVG. Those are the ID values we use in these settings to define the individual elements for each hand and dial.

If we include app.js in our index.html file and open it, we should have both dials ticking along with the current time.

It is very likely, though, upon refreshing our page that the hands are not rotating the way we envision - from the center point of the dial. We will need to define the transform-origin point of these elements.

app.css 

svg {
  height: 244px;
  width: 244px;
}

#primary-second-hand,
#primary-minute-hand,
#primary-hour-hand {
  transform-origin: 122px 122px;
}

#secondary-second-hand,
#secondary-minute-hand,
#secondary-hour-hand {
  transform-origin: 122px 196px;
}

#power-reserve-hand {
  transform-origin: 122px 70px;
}

In the code above, we set the SVG to a specific size. This allows us to more easily define specific pixel transform-origin values. It would be just as easy to use keywords such as bottom center for some elements but for better browser support, specific values are encouraged.

Let's include our new stylesheet into index.html and move back to app.js.

Now, back in app.js let's play with a couple additional settings for our secondary dial.

app.js 

const settings = {
  dials: [...
  {
    name: 'secondary',
    hands: {
      hour: 'secondary-hour-hand',
      minute: 'secondary-minute-hand',
      second: 'secondary-second-hand'
    },
    offset: '+4',
    sweep: true
  }]
};

In the updated settings, we've added a GMT offset value of '+4' hours and set the sweep property on the secondary second hand to offer contrast from the ticking second hand on the primary dial.

If all looks well so far, let's finish this up by adding the power reserve. 

const settings = {
  ...
  reserve: {
    id: 'power-reserve-hand',
    range: [-90, 90]
  }
};

Again, we supply the ID of the hand element to the settings. Then we define the range of motion for the power reserve, which is relative to its originally drawn position. The hand here is originally drawn at zero degree in the middle of the indicator. In order for the hand to reach either the full or empty ends of the indicator it will need to rotate 90 degrees either way. So we set the range to [-90, 90] to define the minimum and maximum values.

So the updated app.js will look like this: 

(function() {
  "use strict"
  const Watch = require('ticktock-js');
  const settings = {
    dials: [{
        name: 'primary',
        hands: {
          hour: 'primary-hour-hand',
          minute: 'primary-minute-hand',
          second: 'primary-second-hand'
        }
      },
      {
      name: 'secondary',
      hands: {
        hour: 'secondary-hour-hand',
        minute: 'secondary-minute-hand',
        second: 'secondary-second-hand'
      },
      offset: '+4',
      sweep: true
      }
    ],
    reserve: {
      id: 'power-reserve-hand',
      range: [-90, 90]
    }
  };
  let firstWatch = new Watch(settings);
})();
